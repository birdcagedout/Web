CSS font-size property

폰트 크기를 변경하려면 css의 font-size 속성을 사용하면 됩니다.

용어 설명

px(픽셀)
px는 웹에서 사용되는 가장 기본적인 크기 단위입니다.
rem
rem은 html(최상위) 태그의 폰트 크기(기본=16px)를 기준으로 배수에 해당하는 크기를 표현하는 단위입니다.
em
em은 부모 태그의 폰트 크기의 배수 단위입니다.
vw
vw(viewport width)는 화면 가로 크기 단위입니다. 화면 가로크기는 100vw입니다.
vh
vh(viewport height)는 화면 세로 크기 단위입니다. 화면 세로크기는 100vh입니다.
